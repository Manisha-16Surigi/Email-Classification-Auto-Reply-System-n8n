# Intelligent Email Classification & Auto-Reply System (n8n)

## Project Overview
This project is an AI-powered Gmail automation built using n8n.  
It automatically classifies incoming emails and creates draft replies using OpenAI.

## Features
- Gmail trigger for new emails
- AI-based email classification
- Automatic Gmail labels
- Auto-draft replies for different email types
- Fraud / suspicious mail detection

## Email Categories
- Strong drafts
- Normal emails
- Suspicious emails
- Marketing & Promotions

##  Tech Stack
- n8n (Cloud)
- Gmail API
- OpenAI (Chat Model)
- AI Agent
- Switch Node
